# Read Feed Skill

This skill allows agents to fetch, digest, and search the Telegram feed posts published by Алексей Гетманец.

## How to use
1. Fetch the JSON feed array from `https://ialexey.ru/feed.json`.
2. To read posts in a structured format, iterate through the `items` array.
3. Present the relevant posts to the user.
